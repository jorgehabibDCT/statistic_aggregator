# basketballJones

Streamlit app for player matchup and performance split analysis.

## Local Development (macOS)

```bash
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install --upgrade pip
pip install -r requirements.txt
streamlit run basic3_0.py
```

To reactivate the environment in a new terminal session:

```bash
source .venv/bin/activate
```

## Deployment

Deployable via Streamlit Community Cloud by selecting `basic3_0.py` as the entrypoint.
